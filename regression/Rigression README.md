# 📊 Advertising Sales Prediction

## 📌 Project Description
This project predicts sales based on advertising spend on TV, Radio, and Newspaper using Machine Learning.

## 🚀 Features
- Linear Regression model
- Random Forest Regressor
- High accuracy prediction
- Simple UI (Flask / Streamlit)

## 📂 Dataset
- TV
- Radio
- Newspaper
- Sales (target)

## 📈 Model Performance
- R² Score: 0.88+

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Flask / Streamlit

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
